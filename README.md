# Mahir Dalal - Personal Portfolio

Welcome to my personal portfolio website! This repository contains the source code for my interactive portfolio, built with a modern tech stack to showcase my competitive programming achievements, projects, and technical skills.

## Live Website
Check out the live portfolio: [mahirdalal.me](https://mahirdalal.me)

## Features
- **Interactive UI**: A fully responsive, terminal/bento-grid style design that looks stunning across devices.
- **Live CP Stats**: Automatically fetches real-time rating and problem-solving statistics from LeetCode and Codeforces.
- **Embedded Resume**: View and download my professional CV directly from the interface.
- **Projects Showcase**: Highlights my key work including descriptions, tech stacks, and source links.

## Tech Stack
- **Framework**: React.js / Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React / Custom SVG Assets
- **Deployment**: Vercel

## Run Locally

**Prerequisites:** Node.js installed on your machine.

1. Clone the repository:
   ```bash
   git clone https://github.com/mahirdll29/portfolio-website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio-website
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
